gRPC client in nodejs
===========================

PREREQUISITES
-------------

- `node`: This requires Node 0.12.x or greater.     
- `grpc` and `grpc-tools`:     

INSTALL
-------

   ```sh
   $ npm install
   ```

TRY IT!
-------

 - Know in which host and port number is the server running. Run the client

   ```sh
   $ # from this directory
   $ node route_guide_client.js --db_path ./route_guide_db.json 
   ```
